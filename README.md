

## Overview
A machine learning-based sentiment analysis application that classifies text into Positive, Negative, or Neutral sentiments.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- TF-IDF Vectorization
- Logistic Regression

## Dataset
Twitter Entity Sentiment Analysis Dataset

## Model Accuracy
93%

## Features
- Sentiment Prediction
- Confidence Score
- Probability Distribution
- Streamlit Web Interface

## Project Structure
app/
dataset/
model/
train_model.py
main.py

## How to Run

### Train Model
python train_model.py

### Run Application
python -m streamlit run app/app.py
